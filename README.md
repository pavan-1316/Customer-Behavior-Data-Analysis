# Customer Behavior Data Analysis

## Project Overview
The Customer Behavior Data Analysis project is focused on analyzing customer shopping patterns to extract meaningful business insights. The project uses Python, SQL, and Power BI to clean, process, analyze, and visualize customer transaction data.

The main goal is to understand customer demographics, purchasing behavior, product preferences, and the impact of discounts and promotions on sales. These insights help businesses make data-driven decisions to improve marketing strategies, increase revenue, and enhance customer satisfaction.

---

## Objectives
- Analyze customer purchasing behavior and spending patterns
- Identify top-selling products and categories
- Understand customer demographics (age, gender, etc.)
- Evaluate the impact of discounts and promo codes
- Measure customer satisfaction using review ratings
- Build an interactive dashboard for decision-making

---

## Dataset Description
The dataset includes customer shopping details such as:

- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount
- Location
- Size
- Color
- Season
- Review Rating
- Subscription Status
- Payment Method
- Shipping Type
- Discount Applied
- Promo Code Used
- Previous Purchases

- Customer demographics

- Revenue KPIs
